練習問題（オブジェクト編）

問題1: 自分のプロフィールを作ろう

次のようなオブジェクト myProfile を作成してください。

  • プロパティ

    • name: あなたの名前（文字列）
    • age: あなたの年齢（数値）
    • hobby: 趣味（文字列）

その後、console.log() を使って「名前」と「趣味」を出力してください。

問題2: 商品の合計金額を計算しよう

次のオブジェクトを作成してください。

let item = {
  name: "りんご",
  price: 120,
  quantity: 5
};

このオブジェクトを使って、合計金額（価格 × 個数）を計算し、console.log()で出力してください。

問題3: メソッドを作ろう

次のような dog オブジェクトを作成してください。

  • name: 犬の名前（文字列）
  • age: 年齢（数値）
  • bark: 「ワンワン！」と表示するメソッド（関数）

その後、dog.bark() を呼び出して実行してください。

問題4: オブジェクトの中のオブジェクト

次のようなオブジェクトを作ってください。

let student = {
  name: "太郎",
  score: {
    math: 80,
    english: 90
  }
};

ここから 英語の点数 を取り出して console.log() で表示してください。

問題5: 配列とオブジェクトの組み合わせ

次のような配列を作ってください。

let fruits = [
  { name: "りんご", price: 100 },
  { name: "みかん", price: 80 },
  { name: "ぶどう", price: 200 }
];

この配列の中から「みかんの値段」を取り出して表示してください。
